# Quien_Quiere_Ser_Millonario
Juego en python con Pygame de "¿Quien quiere ser Millonario?

## Instalación

1. Clona el código:
```bash
git clone https://github.com/AndreRivas1103/Quien_Quiere_Ser_Millonario.git
cd Quien_Quiere_Ser_Millonario
```

2. Instala las dependencias:
```bash
pip install -r requirements.txt
```

O instala pygame directamente:
```bash
pip install pygame
```

## Cómo jugar

1. Corre el código:
```bash
python juego/quien_quiere_ser_millonario.py
```

2. Ingresa tu nombre en la consola

3. Selecciona alguna de las diferentes respuestas con el teclado:
   - **A, B, C, D**: Seleccionar respuesta
   - **Enter**: Confirmar respuesta
   - **Z**: Usar ayuda 50/50 (elimina 2 respuestas incorrectas)
   - **X**: Cambiar de pregunta

4. Al finalizar el juego te dará el resumen de la partida en consola
